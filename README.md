# Арматурина

Мега смешной Telegram-бот для групп. Работает только с админами: если админ пишет фразу со словом `Арматурина` и `@username`, бот отвечает рандомным стикером или GIF на последнее сообщение указанного пользователя.

## Запуск

1. Создай бота через BotFather и выключи ему privacy mode: `Bot Settings` -> `Group Privacy` -> `Turn off`.
2. Скопируй `.env.example` в `.env` и укажи `ARMATURINA_BOT_TOKEN`.
3. Установи зависимости и запусти:

```bash
npm install
npm start
```

Если в одном контейнере запускается несколько ботов, можно импортировать Арматурину в главный ESM-файл:

```js
import { armaturina } from './armaturina/src/index.js';

await armaturina();
```

`armaturina()` не блокирует запуск остальных ботов: она поднимает polling в фоне и возвращает инстанс после загрузки файла с пулом.

Если не хочешь использовать env-переменные, передай настройки явно:

```js
import { armaturina } from './armaturina/src/index.js';

await armaturina({
  token: process.env.ARMATURINA_BOT_TOKEN,
  dataFilePath: './armaturina/data/media-pool.json'
});
```

## Команды и фразы

Все команды ниже работают только в группе и только от администратора.

- `Арматурина, дай по хрептине @username`
- `Арматурина, @username`
- Ответ на сообщение: `Арматурина`
- Ответ на сообщение: `Арматурина фас`
- `/arm_help` - помощь.
- `/pool` - размер текущего пула.
- `/addstickerpack <pack_name>` - добавить стикерпак по имени.
- `Арматурина, добавь стикерпак <pack_name>` - то же самое обычной фразой.
- Ответить на стикер: `Арматурина, добавь стикерпак` - добавит пак этого стикера.
- Ответить на GIF/animation: `Арматурина, добавь гифку` или `/addgif` - добавит GIF.

## Хранение

Пул стикеров/GIF хранится в JSON-файле `data/media-pool.json`. После добавления стикерпака или GIF бот сразу сохраняет файл, поэтому перезапуск процесса не очищает пул.

Путь можно поменять в `.env`:

```bash
DATA_FILE=data/media-pool.json
```

Последние сообщения пользователей хранятся только в памяти процесса. После рестарта бот снова начнет запоминать последние сообщения по мере активности в группе.
